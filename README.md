PROJETO SAAS Desenvolvido pela ideia da matéria de Experiência Criativa
do curso de Engenharia de Software da PUCPR.

O projeto é um SAAS que presta serviços para melhorar o desempenho de áreas de empresas.
A ideia é oferecer o serviço com preço de mensalide fixa. Disparamos formulários para equipes de empresas clientes, informamos cada equipe semanalmente a preencher os formulários com perguntas de feedback do trabalho prestado (ex: carga horária respeitada? Tratamentos abusivos?), todas as respostas obviamente anônimas, sem exposição das respostas. 
Faríamos um leaderboard das equipes que mais respondem aos formulários e as equipes que mantiverem essa frequência mensalmente seriam bonificadas com produtos ofericos das empresas que prestamos serviço. (Ex: Empresa de Software - Assinatura de streaming, empresa de roupa - roupas novas e com desconto exclusivo).
